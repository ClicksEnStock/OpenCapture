OpenCapture Object - Expressions

Last Error

Returns the last error encountered by the capture device. Once read the error is erased and can not be retrieved anymore.
Note that if several errors happened before this string is read, only the last error is provided.

Track color menu

Get Color Position X from Id

Retrieves the position X from a color object defined by the [Set Tracking Color range in HSV] action.
If no color object is defined, returns an invalid value (-1000).

This action takes 1 argument:
  • Color Id: choose which color you want to get its position. Set 1 for first color or 2 for the second one.

Get Color Position Y from Id

Retrieves the position Y from a color object defined by the [Set Tracking Color range in HSV] action.
If no color object is defined, returns an invalid value (-1000).

This action takes 1 argument:
  • Color Id: choose which color you want to get its position. Set 1 for first color or 2 for the second one.

Get HSV color from positions

Retrieves a color in HSV format at a defined position on the current frame.
This action takes 3 arguments:

  • Position X: set the position X of the color to retrieve.
  • Position Y: set the position Y of the color to retrieve.
  • Color Index: color index in the HSV format (Hue=1, Saturation=2, Value=3).
Note that the positions are relative to the camera object position (upper left corner) and not to the position in the scene.

Get RGB color from positions

Retrieves a color in RGB format at a defined position on the current frame.
This action takes 3 arguments:

  • Position X: set the position X of the color to retrieve.
  • Position Y: set the position Y of the color to retrieve.
  • Color Index: color index in the RGB format (Red=1, Green=2, Blue=3).
Note that the positions are relative to the camera object position (upper left corner) and not to the position in the scene.

Get Track Color Object Width

Retrieves the rectangle width of the color object currently tracked.
This action takes 1 argument:

  • Color Id: choose which color from tracked colors you want to get its width. Set 1 for first color or 2 for the second one.

Get Track Color Object Height

Retrieves the rectangle heigth of the color object currently tracked.
This action takes 1 argument:

  • Color Id: choose which color from tracked colors you want to get its height. Set 1 for first color or 2 for the second one.

Track object menu

Get Object Position X

Retrieves the position X from the object desribed by the classifier file.
If no classifier file was provided, returns an invalid value (-1000).

Get Object Position Y

Retrieves the position Y from the object desribed by the classifier file.
If no classifier file was provided, returns an invalid value (-1000).

Get Object Width

Retrieves the width from the object found.
If no classifier file was provided, returns an invalid value (-1000).

Get Object Height

Retrieves the height from the object found.
If no classifier file was provided, returns an invalid value (-1000).

Track feature menu

Get number of good matches

Retrieves the number of keypoints found which matches between feature object and camera image.

Get Corner Position X

Retrieves the position X from the object desribed by the classifier file.
If no classifier file was provided, returns an invalid value (-1000).

Track barcode menu

Barcode Text

Retrieves the string corresponding to the last barcode found.
