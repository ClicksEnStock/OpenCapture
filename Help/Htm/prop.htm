OpenCapture Object - Properties

Main Parameters

Flip camera image (horizontal)

Checkbox to choose if the camera image is flipped. This setting is per default unchecked, that is to say without flipping

Background color

Set the default background color if there is no preview or/and capture.

Debug

Show tracking objects.

  • for color tracking: show the rectangle on the color object found. It displayed both tracked colors even if no or only one color is defined.
  • for object (face) tracking: show an ellipse on the object found.
  • for feature object tracking: show a polygon (4 corners) on the object found.
