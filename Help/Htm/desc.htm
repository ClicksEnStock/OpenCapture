OpenCapture Object

The OpenCapture object deals with video capture devices. It can capture video stream, display the rendering on the screen (preview), track color objects, track objects with a classifier file, track feature object, track QR code or barcode and save camera image or video to files. It is using the OpenCv library 2.4.8. to capture and process video stream.
